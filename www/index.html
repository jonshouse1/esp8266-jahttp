ESP8266 Flash memory HTTP server

Welcome to the ESP8266 (ESP-12 Or similar) web server demo.

Acting as a static web page server with a filesystem in flash is pushing it a little for these modules.

I wrote this page with vi, nobody said I was a web designer.

Initially I had problems making TCP work on this module. I am more used to vanilla sockets or write() interface rather than the mandated callbacks.

The basic HTTP server is based on the well structured TCP server demo code by Tom Trebisky:
tcp_server.c Original link http://cholla.mmto.org/esp8266/OLD/sdk/tcp_server.c

I modified the TCP server to make my own demo code, that spat out a continuous stream of text in response to a telnet connection:

The memory filesystem code is a slightly modified version of this code:
mfs.c mfs.h Taken from this project : https://github.com/cnlohr/esp8266ws2812i2s

I am using the ws2811 project above for some of my Xmas lights, very impressive work.

While trying to fix the dns server I knocked up I did a lot of google searches for various esp API calls, one of these was a DNS resolver that was frankly much better looking than mine, so that is now included in this project.
